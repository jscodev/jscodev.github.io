# docker

# 安装docker

  1. yum install -y yum-utils 安装需要的依赖
  2. yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo 配置国内淘宝镜像源
  3. yum makecache fast 更新yum软件包索引
  4. yum install docker-ce docker-ce-cli containerd.io 安装docket 社区版：docket-ce 企业版：docket-ee
  5. systemctl start docker 启动docker
  6. docker version 查看是否安装成功
  7. docker images 查看下载的hello-word镜像

yum remove docker-ce docker-ce-cli containerd.io 卸载docker依赖 rm -rf /var/lib/docker 删除资源 (/var/lib/docker是docker的默认工作路径)

PS:中途需要你接受一下密钥和安装包的提示，选择是即可

附带官方安装步骤 (opens new window)

# 下载容器

  • docker search centos 查找centos的镜像（这一步可以直接忽略，默认下载最新的）
  • docker pull centos 或者 docker pull centos:指定版本号tag 下载镜像（这里推荐直接使用阿里云容器镜像加速）

阿里云镜像加速步骤：

  1. 登录阿里云
  2. 进入控制台
  3. 在侧边栏中选择产品与服务点击里面的容器镜像服务
  4. 进到容器镜像服务中点击左侧边栏的镜像加速器
  5. 配置 /etc/docker/daemon.json 把大括号里面的内容复制进去并保存
  6. 依次执行命令 systemctl daemon-reload systemctl restart docker

# 运行centos

  • docker run -it centos 以交互模式运行容器
  • docker ps 查看运行中的容器
  • exit 退出容器并停止运行
  • Ctrl + P +Q 退出容器不停止运行
  • docker exec -it 容器id bashShell 进入运行中的容器开启新的终端（常用）
  • docker attach 容器id进入运行中的容器不会启动新的进程

# 将容器内的文件拷贝到主机

退出容器执行命令docker cp 容器id:容器内路径 目标路径(这里容器运不运行都可以拷贝)

# docker容器内无法联网问题

centos8解决方案：

firewall-cmd --zone=public

--add-masquerade --permanent

firewall-cmd --reload

systemctl restart docker
