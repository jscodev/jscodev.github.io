                        # 配置golang

                        在vscode中打开go文件会让你下载install all，这个时候需要配置代理否则会下载失败。

                        • GO111MODULE=off 无模块支持，go会从GOPATH和vendor文件夹寻找包。(也就是GOPATH模式)

                        • GO111MODULE=on 模块支持，go 会忽略GOPATH和vendor文件夹，只根据go.mod下载依赖。(也就是GOMODULE模式)

                        • GO111MODULE=auto 在$GOPATH/src外面且根目录有go.mod文件时，开启模块支持。

                        go env -w GO111MODULE=on go env -w GOPROXY=https://goproxy.cn,direct

                        • GOSUMDB 校验拉去第三方库是否完整
                        • GOPRIVATE 设置私有仓库，设置后不会通过GOPROXY拉取和GOSUMDB校验

                        # 字符串

                        Go语言中字符串使用双引号包裹的，单引号里面是字符。

                        一个英文字符表示1个字节，一个中文汉字表示3个字节（1字节=8bit）。

                        格式化：

                        • %v,原样输出
                        • %T，打印类型
                        • %t,bool类型
                        • %s，字符串
                        • %f，浮点
                        • %d，10进制的整数
                        • %b，2进制的整数
                        • %o，8进制
                        • %x，%X，16进制 - %x：0-9，a-f - %X：0-9，A-F
                        • %c，打印字符
                        • %p，打印地址

                        # fmt

                        
var a = 10

// 查看类型
fmt.Printf("%T\n",a)
// 查看值
fmt.Printf("%v\n",a)

                        fmt.Printf和fmt.Println区别

                        Println有自动换行操作而Printf没有，但是Printf有占位符的处理，用Println做拼接时会有好多空格，所以还是用Printf做拼接。

                        # 数组的循环

                        使用range 不需要操作下标，每次循环自动获取元素中的下标和对应的值。如果到达数组的末尾，自动结束循环。

                        
arr := [5]int{1,2,3,4,5}
//range方式循环数组
for index,value:=range arr{
    fmt.Println(index,value)
}
// 使用for循环
arr := [5]int{1,2,3,4,5}
//for循环
for i:=0; i<len(arr);i++{
    fmt.Println(arr[i])
}

                        # 管道

                        创建管道使用chan关键字

                        
a := make(chan int) // 无缓冲
a := make(chan int,10) // 带有缓冲

                        注意点

                        1. 主进程阻塞会造成程序崩溃，协程阻塞会造成内存泄漏
                        2. 每次读写完管道需要使用close关闭管道或者exit退出协程，并且不能重复关闭一个已经关闭的协程
                        3. 如果管道没有使用make分配空间时，默认值为nil
                        4. 关闭管道的动作要写在写端
                        5. 读写次数一定要对等
                        6. 向一个已经关闭的管道写数据会崩溃，读不会但是会返回0值

                        # for range

                        循环管道时最好使用for range的方式，它会自动获取value的值

                        
func main() {
	a := make(chan int, 10)
	go func() {
		for i := 0; i < 10; i++ {
			a <- i
		}
		close(a)
	}()

    // 判断管道是否关闭

	for {
		v, i := <-a
		fmt.Println(v, i)
		if !i {
			fmt.Println("管道已经关闭")
			return
		}
	}
}

                        # sync

                        • Mutex: sync.Mutex称为互斥锁，常用在并发编程里面。互斥锁需要保证的是同一个时间段内不能有多个并发协程同时访问某一个资源(临界区)；sync.Mutex有2个函数Lock和UnLock分别表示获得锁和释放锁。

                        • RWMutex：读写锁

                        • WaitGroup：并发等待组

                        • Once：执行一次

                        • Cond：信号量

                        • Pool：临时对象池

                        • Map：自带锁的map

                        
// RWMutex：读写锁
// 写操作
func (rw *RWMutex) Lock()
func (rw *RWMutex) Unlock()

// 读操作
func (rw *RWMutex) RLock()
func (rw *RWMutex) RUnlock()

RLocker()能获取读锁，然后传递给其他协程使用。
