            # 初遇mysql

            # 前言

            我们在写后台程序的时候总会选择一个合适的数据库，这里我选择了Mysql，至于为什么...这个东西比较符合我的要求，Oracle大型了一些没必要，还有许多数据库包括Redis、TitanDB、Neo4j、Mongodb···大家可以自行了解一下，我这就不一一列举了，下面有请我们的靓仔mysql登场。

            # 安装mysql

            这里以win10举例，首先去官网下载 (opens new window)，下载后找个盘符新建文件夹Mysql，然后把下载好的zip文件放到里面解压出来，因为新版是可以不用配置my.ini文件，所以这里直接去配置环境变量就好了，右键此电脑，或者文件夹左侧右键也可以，咋方便咋来就完事了。

            点击高级系统设置，再点击环境变量，在系统变量里面新建一条路径，路径指向你解压安装包的路径一直到bin，然后在Path变量里面也加上相同路径，如果没有新建一个Path。

            # 初始化数据库

            这个时候就到了关键的地方了，也是大多数人卡住的地方，所以这里你好好看，跟上我的步骤，包你一路小平坡。

            • 首先以管理员身份打开命令行工具
            • 然后执行cd /d 你的安装目录\bin
            • 进到bin目录后执行mysqld --initialize --console最后的console是为了打印一下执行日志，也可以不加
            • 这个时候你可能会报一个错误‘找不到vcruntime140.dll’，没关系这里给你准备了下载链接 (opens new window)，点进去下载后安装重启即可
            • 重启完回来重新执行上面步骤，初始化完成时会出现一个root@localhost: HSUshitaW+-*/这个冒号后面就是你的数据库初始密码，用户名是root，可别搞丢了，后面要用的
            • 继续执行mysqld --install 你的数据库名这里库名拿mysql举例
            • 设置完后执行net start mysql，出现服务启动成功的话就代表你已经安装好了，使用net stop mysql可以关闭它
            • 接下来改密码，执行mysql -uroot -p这时候会让你输入密码，你就把上面输入的密码拿过来输入就好了，下一步执行ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123';这里记得不要落掉结尾的分号，BY后面就是你要设置的密码

            以后打开数据库都需要用管理员身份打开，不然会报拒绝访问的错误，嫌麻烦可以在桌面上创建一个快捷方式，到这里我们就可以和mysql愉快的玩耍了！🎉

            # 使用nodejs操作mysql

            
// npm安装依赖后引入
const mysql = require("mysql");
//创建连接
var connection = mysql.createConnection({
  host: 'localhost', // 连接的主机ip
  user: 'admin', // 账号
  password: '123456', // 密码
  database: 'database', // 连接的数据库
  multipleStatements: true // 开启多查询语句模式
})
// 如果err是null就代表连接成功
connection.connect(err=> {
  console.log(err);
})
// 查询语句
let sql = `SELECT * FROM login`
// 执行查询，格式化数据
connection.query(sql, (err, res)=> {
  let data = JSON.parse(JSON.stringify(res))
  console.log(data[0]);
})

            关闭数据库的两种方式：

            connection.end 执行之前会把所有操作语句执行完

            connection.destroy 立即关闭

            # 连接池

            // 创建数据库连接池
var pool = mysql.createPool({
  host: 'localhost', // 连接的主机ip
  user: 'admin', // 账号
  password: '123456', // 密码
  database: 'database', // 连接的数据库
  connectTimeout: 100, //连接超时时间
});
// 在连接池中获取缓存的连接
pool.getConnection((err, connection) => {
  if (err) throw err;
  let sql = `SELECT * FROM login`;
  connection.query(sql, (err, res) => {
    if (err) throw err;
    console.log(res);
  });
  connection.release(); // 归还连接
});
