# 写在开始的话

相信我们都曾经历过一段孤军奋战的生活，那段日子很难熬，无数次的打击让人想逃避、想退缩，都说陪伴才是最长情的告白，那最爱我们的应该就是Hello Wrod了，从毕业到工作再到现在，从一开始对产品和领导的无理要求言听计从，每天面对着繁重的业务代码压得人喘不过气，到现在的手拿把掐（吹牛呢🤣），工作也开始不再匆匆忙忙没有什么个人时间，所以就在最近两天突然想搭建一个自己的博客网站，随时记录自己的所思所想，也希望可以结识一些志同道合的伙伴。

# 关于此站

因为不想把一个简单的东西搞得太复杂，所以这里不做太多花哨的特效~

这个博客站主要是分享一些个人见解，和记录一些平时的笔记、遇到的难题还有一些工作上的感悟。希望可以和大家一起学习，共同进步。

# 写在结尾的话

此博客将不断更新维护，有意见可以提出来。
